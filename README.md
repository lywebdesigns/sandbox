## Quick Start Guide
1.  Clone the repository locally to your computer and open it in VS Code.
2.  Run `npm install` to install all dependencies.
3.  After the installation is complete, run `npm start` to start the development server.
4.  Fill out the `./src/_data/client.js` file with the appropriate information for your client.
5.  Adjust the `:root` variables in `./src/assets/[sass or less]/root.[scss or less]`
6.  Modify the website files (use `./src`, NOT `./public`) as needed. Use the template file in `content/pages/_template.txt` to get started, or modify the existing pages.
7.  Deploy using your preferred hosting provider.
